# SNS_data_analysis
소셜네트워크서비스(SNS)를 활용한 데이터 분석하기!


coinpan_data_analysis : 비트코인, 블록체인 등의 주제인 오픈 커뮤니티 코인판(coinpan)을 텍스트 데이터를 활용해서 메디블록(mediblock) 분석
